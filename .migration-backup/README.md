# ⚡ TYSM Daily Faucet Mini-App

A gamified Web3 SocialFi Mini-App built for the Farcaster ecosystem, deployed on the Base network. This application incentivizes community engagement through a structured daily token distribution loop.

## 🚀 Key Features

*   **⚡ Share-to-Unlock:** Users must share (Cast) their current streak status to the Farcaster network before unlocking the on-chain claim action, ensuring a 100% viral loop.
*   **🎯 Progress Bar & Milestones:** Visual indicator tracking the user's progress through the 30-day streak cycle.
*   **🎁 Milestone Badges:** Special visual rewards unlocked on major reward milestones (Days 7, 15, and 30).
*   **⏳ Countdown Timer:** Keeps users motivated by showing exactly how many days remain until the next major reward tier.

---

## 📊 Gamified Reward Structure (30-Day Cycle)

The faucet utilizes a dynamic distribution model to prevent "hit-and-run" claims and maximize user retention:

| Day Period | Daily Reward Amount | Notes |
| :--- | :--- | :--- |
| **Days 1 - 6** | 2,000 $TYSM / day | Base Streak Setup |
| **Day 7** | **10,000 $TYSM** 🥉 | **Milestone Bonus 1** |
| **Days 8 - 14** | 2,000 $TYSM / day | Mid-Cycle Retention |
| **Day 15** | **40,000 $TYSM** 🥈 | **Milestone Bonus 2** |
| **Days 16 - 29**| 2,000 $TYSM / day | Late-Cycle Engagement |
| **Day 30** | **90,000 $TYSM** 🥇👑 | **Ultimate Milestone** |
| **Day 31+** | *Reset* | Cycles back to Day 1 |
